# AI Usage Log

## Overview
This document outlines how AI tools (primarily ChatGPT) were used in the development of this sentiment analysis project. All final decisions, code execution, and evaluations were completed and verified by the author.

---

## AI Tools Used
- ChatGPT (OpenAI)

---

## Purpose of AI Assistance

AI was used as a **support tool** to enhance productivity, clarify concepts, and assist with structuring the project. Specifically, AI helped with:

### 1. Code Guidance
- Suggested structure for machine learning pipeline (EDA → Cleaning → Feature Engineering → Modeling → Evaluation)
- Provided examples for:
  - TF-IDF vectorization
  - Logistic Regression and Naive Bayes models
  - GridSearchCV for hyperparameter tuning
  - Classification metrics (accuracy, F1-score, confusion matrix)

### 2. Debugging Support
- Helped identify and fix errors such as:
  - Undefined variables (e.g., `X_train`)
  - Incorrect pipeline order
  - Missing predictions for evaluation

### 3. Markdown & Documentation
- Assisted in writing:
  - Section explanations
  - Model evaluation summaries
  - README file
  - Structured notebook sections aligned with assignment rubric

### 4. Project Structuring
- Helped organize the notebook into clearly defined steps:
  - Step 1: Data Collection
  - Step 2: EDA
  - Step 3: Text Cleaning
  - Step 4: Feature Engineering
  - Step 5: Model Training
  - Step 6: Evaluation & Comparison
  - Step 7: Custom Inference

---

## What AI Did NOT Do

- AI did **not** independently complete the assignment
- AI did **not** make final decisions on:
  - Model selection
  - Interpretation of results
  - Evaluation and reflection
- AI did **not** generate the final conclusions

All outputs were:
- Reviewed
- Modified when necessary
- Tested in Google Colab
- Interpreted by the author

---

## Custom Inference & Reflection

The **Step 7 evaluation and reflection** was completed independently.  
The author:
- Created custom test examples
- Defined expected labels
- Compared predictions vs. personal judgment
- Analyzed model generalization ability

This project complies with academic integrity guidelines. AI was used as a **learning aid and assistant**, not as a replacement for original work.

All final code, analysis, and conclusions represent the author's own understanding and effort.
